Facial Recognition Attendance

Real-time attendance system using OpenCV + face_recognition with automatic CSV logging via Flask interface.

PythonFlaskOpenCVface_recognition
Source Code Portfolio

Highlights

  • Live webcam face matching against known encodings
  • Session attendance logging with timestamps
  • Simple onboarding: add images in imageAttendance/

Run locally

This project requires local webcam + Python runtime. GitHub Pages hosts documentation only.

pip install -r requirements.txt

python app.py then open http://127.0.0.1:5000/mark_attendance_page
